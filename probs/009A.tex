
\prob{009A}{巧解四次方程II}

求方程
\[ (6x + 7)^2(3x + 4)(x + 1) = 6 \]
的实数根。
\problabels{yellow/代数, green/方程相关问题}

\textit{JYH提供的题目。}

\ans{$x = -5/3$或$x = -2/3$}

\subsection{换元}

两边同乘3，得
\[ (6x + 7)^2(3x + 4)(3x + 3) = 18 \]
令$p = 3x + 4, q = 3x + 3$，则
\[ pq(p + q)^2 = 18, p - q = 1 \]
于是
\[ pq(4pq + 1) = 18 \]
解关于$pq$的二次方程，得$pq = 2$或$pq = -9/4$。

若$pq = 2$，则$p = -1, q = -2$或$p = 2, q = 1$，此时$x = -5/3$或$x = -2/3$；若$pq = -9/4$，此时不存在实数$p, q$满足方程组。

综上，$x = -5/3$或$x = -2/3$。
